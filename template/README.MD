#### 启动开发环境
```
$ yarn dev
```

#### 调试服务器端 js
首先启动调试项目
```
$ yarn debug
```
使用 vscode，左侧“调试”菜单，点击 【Attach(projectName)】，就可以愉快的在vscode里调试服务器端代码了

#### 生成打包并使用 pm2 启动项目
```
$ yarn pm2start
```
